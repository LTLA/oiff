◆ last

Last (i.e., most relaxed) filter threshold that achieves the maximum number of discoveries. If OptimizeFilter::retain_larger is false, this is the largest filter threshold; otherwise it is the smallest.

◆ middle

template<typename F >
F oiff::OptimizeFilter::Thresholds< F >::middle = 0

An intermediate filter threshold that achieves the maximum number of discoveries. This is usually somewhere between first and last, and is close to the median of all filter thresholds that achieve maximum discoveries. (It is exactly the median if the number of candidate thresholds is odd; otherwise it is the closest threshold that is more relaxed than the median.)
