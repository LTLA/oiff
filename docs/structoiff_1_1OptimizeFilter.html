Detailed Description

Optimize the filter threshold to maximize discoveries.

This class contains the core functionality in the oiff library. Methods are provided to identify the filter threshold that maximizes the number of discoveries at a nominal FDR threshold. This can also be done via rounds of subsampling to mitigate loss of FDR control from aggressive inclusion of discoveries.
